# Simple Ads Site

<img src="https://github.com/jnetto23/site-de-classificados/blob/master/screenshots/ads_site-home.png" width="100%" height="auto"/>
  
## Objective/Challenge

Create backend/frontend of the a simple ads/classified system.

## Stack Used

- Backend:
  - PHP7
  - MySQL
- Frontend:
  - HTML - Use of semantic tags;
  - CSS3 - Mobile first standard, use of Flexbox and media queries;
  - JavaScript
